【2ヶ月記念 クイズ＆ガチャサイト】

■ ファイル
- index.html
- style.css
- script.js

■ 開き方
index.html をダブルクリックするとブラウザで確認できます。

■ GitHub / Vercelへの反映
1. GitHubリポジトリ内の index.html / style.css / script.js を差し替える
2. 変更をコミットして main ブランチへプッシュする
3. Vercelが自動で再デプロイします

■ クイズの変更
script.js の先頭にある quizzes 配列を編集してください。
answerIndex は、正解の選択肢番号を0から数えて指定します。
例：2つ目の選択肢が正解なら answerIndex: 1

■ 景品の変更
script.js の prizes 配列を編集してください。

■ 名前の変更
index.html と script.js 内の「なーちゃん」「たくと」を置換してください。
